Juno is a system for rapidly deploying practice networks for security attack and defense games. It aims to reduce the barrier to entry into cybersecurity games.

The Team

Juno is a self-proposed project for Oregon State University's undergraduate "capstone" class. The team consists of Casey Colley, Arian Ghorbani, Carter MacNab, and Alexander Marx. The project is advised by Dr. Yeongjin Jang.
